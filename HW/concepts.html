What are some key programming concepts?

Programming has a few core concepts that appear in almost every language.

Understanding these basics will help you read and write code effectively.

Variables

Variables are containers that store data values.

Example

int age = 23

"age" names the variable, and "23" tells us age is equal to 23.

Data Types

Data types define what kind of data a variable can hold.

Example

Data Type Data Held
int Integer
string Text
boolean True/False
float Decimal

Loops

Loops allow you to repeat a block of code multiple times.

Examples

Type Function Example
For Loop Repeats a specific number of times Printing the numbers 1 through 5
While Loop Repeats while a condition is true Keep a game running while the player still has lives.

Conditionals

Preset instructions to make decisions.

Examples

Code Explanation Output
if x > 10
print: ("x is greater than 10")
else:
print("x is 10 or less") 		Prints "x is greater than 10" if the value of x is greater than 10.
Otherwise, print the statement "x is 10 or less"
(If x = 15)
